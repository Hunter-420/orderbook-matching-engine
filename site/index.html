Build Phases

The engine was built incrementally. Each phase adds exactly one architectural layer on top of the previous one.

01

In-Memory Core

Price-time priority matching, doubly linked list price levels, the matching loop. Proven correct before any performance work.

OrderNode · PriceLevel · std::map
Read Docs →
02

Memory Pool

Eliminated every OS heap allocation from the hot path. Pre-allocated one million slots with an O(1) free-list. Flat order directory replaces hash map.

MemoryPool · Free-list · vector<uint32_t>
Read Docs →
03

Network Layer

Single-threaded epoll event loop. 14-byte packed binary protocol. TCP_NODELAY and SIGPIPE handling. Zero locks anywhere.

epoll · TCP_NODELAY · #pragma pack
Read Docs →
04

Hardware Isolation

CPU affinity pinning keeps L1/L2 cache warm. Pre-allocated ring buffer captures nanosecond timestamps without any heap allocation.

pthread_setaffinity · TelemetryBuffer · chrono
